想了很久这个板块要做什么用，最后感觉还是分享一下自己的心情或者记录一下自己在工作或者学习中的一些感受吧。
毕竟朋友圈字数有限，而且有些事情也不好公开讲，所以在这里开辟一个专栏，专门记录一下。

第一篇文章想了好久，最后还是写一下春节过后准备离职这件事儿吧。
作为来了深圳之后的第一份工作，本来确实像踏踏实实的起码工作一年以上，毕竟如果频繁换工作，对于以后求职简历的经历也会有一些影响。
其实不可否认的是，在公司这几个月的时间，让我对我自己的位置有了重新的定位，技术上面也有了一定的补充。
但是公司制度混乱，管理混乱，准备过完年以后回去提离职，如果有纠纷，只能采用法律的手段维权了。
一、制度方面:
1. 合同写的工资和实际工资不对等，可能就是为了将来有劳动纠纷的时候，可以少支付赔偿金。
2. 五险一金的比例是按照深圳市的最低标准交的。
3. 公司有的人单休，有的人双休。更滑稽的是入职的时候说没有年终奖，但是有的人发 13个月 的工资，有的人发 12个月 的工资。
4. 我没有经历过，别人经历过，好像是有一段时间只发的试用期的工资，没发全额工资。
二、管理方面:
1. 项目管理混乱
  1) 没有项目经理，所有的项目都是产品经理瞎指挥，日常开发基本上都在处理之前的人留的烂摊子。
  2) 公司里面不管前端还是后端，代码没有注释，接口没有文档，全靠猜，全靠猜，全靠猜。我们是做 ReactNative 的，之前的 iPad 出了一套类似的产品，我们用接口的时候，只能拿过 iPad 来用 Charles 抓包。
  3) 进度没人把控，基本上就是一个人负责一个项目，你做着 A 项目，突然客户有新需求，老板就逼着产品经理晚上出 Version，你就切换到 B 项目，一天到晚切来切去的。
  4) 技术人员水平参差不齐，总共 5个 React Native 开发，两个打酱油的。基本上 SourceTree 就我和另一个伙计整天疯狂输出。其中有一个 React Native 开发的，从我入职到现在，算什么折扣的规则，算了七八个月了，折扣这东西无非就是加减乘除 + if / else，得有多复杂，能比我考研的微积分、微分方程还难？搞笑。
2. 人事坑
  1) 制度上一些欺骗的行为就不说了，比如承诺的一年有 10天 的假期，我和另一个同事同时请的 3天 年假，给他批了，不给我批。就因为之前瞎指派给我的半年多了都没有人动的项目，不能出 Version，就不给批年假。
  2) 年前说是新冠要检测，说是 20 个人组团，检测机构可以上门，120元钱 / 人，据我所知，那个时候全国已经降价，美团随便找一家正规医院没有超过 80元钱 / 人的，这点儿羊毛都得薅。其实几十块钱真的无所谓，你就挑明了是跑腿费也可以啊，但是总是感觉莫名其妙的被欺骗。最后还假惺惺的在群里每个人又发了 10块钱，我也是呵呵了这操作。
